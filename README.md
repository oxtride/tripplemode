# Промпт для создания сайта-портфолио

## Описание проекта
Создайте минималистичный и стильный черный сайт-портфолио для фронтенд-разработчика с современным дизайном.

## Цветовая палитра
- Основной фон: #000000 (черный)
- Акцентные цвета: оранжевый, красный, желтый
- Текст: #FFFFFF (белый)
- Кнопка: красно-оранжевый градиент с черным текстом

## Структура и дизайн

### Фоновые элементы:
- Черный фон (#000000)
- Абстрактные полупрозрачные пятна оранжевого, красного и желтого цветов
- Полупрозрачная сетка поверх фона (например, сетка 10x10px с прозрачностью 20%)

### Верхнее меню (Header):
- Логотип/текст: белый текст с названием/именем
- Кнопка "Contact" в правом углу с красно-оранжевым градиентом
- Цвет текста на кнопке: черный

### Главная секция (Hero):
- Крупный белый текст - tripplemode
- Подзаголовок: "Front-end" или небольшое описание
- Две кнопки:
  1. "Start" - с градиентным фоном (красно-оранжевый)
  2. "Contact" - прозрачная с белой рамкой

### Секция навыков:
- Заголовок "Навыки" или "Технологии"
- Иконки или названия технологий в современном стиле:
  - HTML
  - CSS
  - JavaScript
  - React
  - Vite
  - TailwindCSS
  - TypeScript

### Футер:
- Минималистичный дизайн
- Контактная информация или копирайт
- Возможно ссылки на социальные сети

## Технические требования
- Адаптивный дизайн
- Плавные анимации при наведении
- Современный, чистый код
- Оптимизированная производительность

## Стиль:
- Минимализм
- Современный дизайн
- Высокий контраст
- Элегантная типографика

## Дополнительные пожелания:
- Плавные переходы между состояниями элементов
- Минимальные анимации для улучшения пользовательского опыта
- Четкая визуальная иерархия контента

Пример градиента для кнопок: linear-gradient(135deg, #ff6b35 0%, #f01 100%)
